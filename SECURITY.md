# Security

## Supported Versions

This is an early-stage personal/open-source tool. Security fixes target the current `main` branch.

## Reporting Issues

If you find a security issue, please open a private report if GitHub Security Advisories are enabled, or contact the maintainer directly.

Do not include API keys, private manuscripts, full prompts, or unpublished plot material in public issues.

## Sensitive Files

Never commit:

- `.env`
- API keys
- generated books
- chapter drafts
- prompt previews from real projects
- vector databases
- platform-submitted manuscript text

The default `.gitignore` is intentionally conservative for this reason.
